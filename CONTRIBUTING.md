# Contributing

## How to Contribute

We welcome contributions to the Voice Cleaning Pipeline project!

### Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/voice-cleaning-pipeline.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test thoroughly
6. Commit: `git commit -am 'Add new feature'`
7. Push: `git push origin feature/your-feature-name`
8. Create a Pull Request

### Code Style

- Follow PEP 8 guidelines
- Add docstrings to all functions/classes
- Include type hints where appropriate
- Write meaningful commit messages

### Testing

Before submitting:
- Test with various audio/video formats
- Ensure backward compatibility
- Update documentation if needed

## Bug Reports

Please include:
- Python version
- OS details
- Steps to reproduce
- Expected vs actual behavior
- Error messages/logs
